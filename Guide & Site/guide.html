Setup

Using the shell

To open the shell, simply double-click on Shell.py in the Omega Language folder.
This will open up the shell, which is used to write single-use-single-line code

Working with files

The Tau IDE (Integrated Developement Envirement) comes pre-installed with Omega.
To use it, double-click on Tau.py.
Inside Tau, you can write Omega or python code, save it, and run it.
If a console window does not pop up, raise it to the top
Do not close this window. Tau is tied to the console and if you remove it Tau will shut down
We will be using files in Tau in the rest of this tutorial.

Basic Concepts

wrt()

wrt() is used to print text onto the console.

Example:
wrt("Hello World")
>>> Hello World

You can also pass in variables to wrt(), which we will learn about later

Ints and Strings

There are a total of 6 data types in omega,
the most basic ones being integers (numbers), floats (decimals)
and strings (a bunch of letters).

The strings look like this:
"A bunch of text"
And are used to "string" characters (letters) together.

Integers on the other hand are any collection of non decimal (1.1111) numbers (negative or positive).
Note that "5" is considered a string and 5 is considered an integer.

Floats are Integers with decimals, like so:
-5.55

Basic Arithmetic

You can perform basic arithmetic on integers like so:
number Operator number

Operation Symbol Meaning
+ Addition
- Substraction
* Multiplication
/ Division
^ or ** exponentation (power of)

Lists

Lists are simply some data in packed together.
Lists look like this:
A list of numbers
[1, 2, 3, 4, 5, 6]

You can also have a list of lists if you want.
You can mix data types inside of lists:
For example:
[1, "hi", 2, "hello", 3, [1, 2, 3]]

To get a value from a list simply do:
>>> list < index

Note that the index starts from 0 and not 1.

Example (to get first element of a list):
[1, 2, 3, 4] < 0
>>> 1

Variables

Variables are used to store data.
They are declared in the following way:
var name = value
The value can be of any of the 6 data types: Int, Float, String, List, Function, and BuiltinFunction.

To change the value of a variable simply do like so (using the operator you wish):
var a = 1
var a = a + 1
wrt(a)

>>> 2

Conditions

Conditions are used to run a piece of code only if a condition is or is not true.
We can do this in Omega with the if/elif/else statements.
if is used in the following way:
if data comparison data {
 wrt("hi")
}


For example:
if 5 >= 4 {
 wrt("It works!")
}
>>> It works!

Tip: You can use variables instead of numbers

Combarison Symbol Meaning
== Is equals
!= Is not equals
< Is less than
> Is greater than
<= Is less than or equal to
>= Is more than or equal to

Each of these will return either true or false, depending on the data on either side.

The elif statement is run when the first if statement is false,
and the condition in the elif is true.
Example:
if 5 >= 6 {
 wrt("This will never run")
}
elif 5 != 4 {
 wrt("This will run!")
}


>>> This will run!

You can use as many elifs as you want. You can also put ifs inside ifs.

This is called nesting, though it's generally not a very good idea for readability purposes.

Finally, we have the else statement
The else statement runs if all the other statements (the if and the optional elifs) don't run.
Example:
if 5 >= 6 {
 wrt("This will never run")
}
elif 5 != 4 {
 wrt("This will run!")
}
else {
 wrt("Math is broken")
}

>>> This will run!

Advanced Concepts

Functions

Functions are used to define a block of code to be run later on.
They are declared in the following way:
def function_name (optional arguments to take in) {
 code to be run
}

To call them (or run them), do like so:
function name(optional arguments to pass)

Let's first focus on functions without arguments.
For example, say we have a function that says hello:
def sayHi(){  wrt("Hello!") }

And then we can call it as many times as we want:
sayHi()
sayHi()
sayHi()
>>> Hello!
>>> Hello!
>>> Hello!


But say we wanted to add one to a specific number
To do this we could use a function with parameters or arguments

To code this function we would do something like this:
def addOne(number) {
 wrt(number + 1)
}

addOne(5)

>>> 6

So number is the unknown variable the function is expecting you to pass, so it can add one to it.

You can have as many arguments as you want in a function. Just make sure to pass them all in when you call it!
